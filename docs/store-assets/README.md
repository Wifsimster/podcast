# Store assets — Ondes

Generated by [`scripts/generate-store-assets.py`](../../scripts/generate-store-assets.py)
(Pillow). They mirror the in-app adaptive launcher icon: five rounded
sound-wave bars in white over a diagonal indigo gradient (`#3D5AFE` → `#1A237E`).

| File | Size | Use |
|------|------|-----|
| `icon-512.png` | 512×512 | Play Console "App icon" (hi-res). |
| `feature-graphic-1024x500.png` | 1024×500 | Play Console "Feature graphic". |

Regenerate after any design tweak:

```bash
python3 scripts/generate-store-assets.py
```

The launcher icon itself is defined as vector drawables under
`app/src/main/res/` (`ic_launcher_background.xml`, `ic_launcher_foreground.xml`,
`ic_launcher_monochrome.xml`) — edit those and re-run the script to keep the
store icon in sync.
